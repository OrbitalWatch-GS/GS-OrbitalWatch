# 🛰️ OrbitalWatch

Sistema de monitoramento de detritos espaciais e suporte à decisão para operadores de satélites.

## 📌 Sobre o Projeto

O OrbitalWatch é uma plataforma web desenvolvida para a Global Solution 2026 da FIAP.
A solução monitora em tempo real a proximidade de detritos orbitais e emite alertas
automáticos de risco de colisão, protegendo ativos orbitais avaliados em centenas de
milhões de dólares.

## 🎯 Problema

Existem mais de 27.000 detritos rastreados em órbita terrestre, viajando a 7 km/s.
Um único impacto pode destruir um satélite e interromper serviços críticos como GPS,
internet e previsão do tempo usados por bilhões de pessoas.

## 💡 Solução

Landing Page informativa com:
- Apresentação do problema e da solução
- Stack tecnológica utilizada
- Galeria com slideshow de imagens orbitais
- Quiz interativo sobre a indústria espacial
- Formulário de contato com validação

## 🛠️ Tecnologias

- HTML5 · CSS3 · JavaScript puro
- Google Fonts (Exo 2 + Share Tech Mono)
- Variáveis CSS · Flexbox · Reset CSS
