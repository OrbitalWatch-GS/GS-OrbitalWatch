01 · Problema

O lixo espacial ameaça
tudo que depende de satélites

⚠️

Síndrome de Kessler

Colisões geram novos detritos em cascata, tornando órbitas inacessíveis por décadas e colocando em risco toda a infraestrutura orbital existente.

📡

Serviços críticos em risco

GPS, previsão do tempo, internet e comunicações globais dependem de satélites. Um único impacto pode derrubar serviços usados por bilhões de pessoas.

💸

Custo bilionário

Cada satélite destruído representa perda de US$100M a US$600M. Os seguros orbitais crescem 30% ao ano em resposta ao risco crescente.

🔍

Monitoramento fragmentado

Operadores privados não têm acesso fácil aos dados de rastreamento orbital. As ferramentas existentes exigem equipes técnicas especializadas e custam caro.

02 · Tecnologia

Stack técnica que
move a plataforma

DADOS

NASA Space-Track

API pública com dados TLE de mais de 27.000 objetos rastreados em órbita terrestre, atualizada em tempo real.

DADOS

ESA DISCOS

Catálogo europeu de objetos espaciais com informações detalhadas sobre detritos, fragmentações e eventos orbitais históricos.

BACKEND

Python · Algoritmo de Conjunção

Modelos matemáticos de probabilidade de colisão baseados em funções exponenciais, calculando risco por distância e velocidade relativa.

FRONTEND

Dashboard Web

Interface HTML, CSS e JavaScript puro com mapa orbital 2D, alertas visuais em tempo real e painel de gestão de missões.

EDGE

Arduino · Sistema Embarcado

Simulação de alerta de bordo de nanossatélite: sensor ultrassônico detecta "detritos" e aciona LEDs de status de risco em tempo real.

ENTREGA

SaaS B2B

Plataforma como serviço para operadores de satélites, com planos escaláveis e API REST para integração com sistemas de telemetria.

03 · Objetivos

O que o OrbitalWatch
pretende alcançar

01

Centralizar dados orbitais abertos

Agregar em uma única plataforma os dados de rastreamento da NASA, ESA e INPE, eliminando a necessidade de múltiplas ferramentas técnicas.

02

Automatizar alertas de risco de colisão

Emitir notificações proativas quando a probabilidade de colisão superar 1:10.000, com antecedência suficiente para planejamento de manobra.

03

Democratizar o acesso ao monitoramento orbital

Tornar acessível a operadores de médio porte o que hoje só está disponível para grandes agências governamentais com equipes especializadas.

04

Contribuir com os ODS da ONU

Proteger a infraestrutura orbital que sustenta os ODS 9 (Inovação), ODS 11 (Cidades Inteligentes) e ODS 13 (Ação Climática).
